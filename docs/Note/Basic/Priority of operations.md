# Priority of operations
运算优先级是只当遇到两个运算符，该先运算哪个。当两个运算符是同级别时，我们将从左到右运算，否则优先级高的先运算。

下面我将列出一个表格，说明运算符的优先级：

| 优先级  | 运算符                   | 描述     |
| ---- | --------------------- | ------ |
| 1 级  | （）                    | 括号     |
| 2 级  | $\times$， $\div$，$\%$ | 乘，除，取模 |
| 3 级  | $+$ ，$-$              | 加，减    |
| 4 级  | <<，>>                 | 左移，右移  |
| 5 级  | &                     | 按位与    |
| 6 级  | ^                     | 按位异或   |
| 7 级  | \|                    | 按位或    |
| 8 级  | &&                    | 逻辑与    |
| 9 级  | \|\|                  | 逻辑或    |
| 10 级 | ？：                    | 三元运算符  |
**备注：上图表格级别越小，优先级越高**
